# rpm-header-reader
A simple tool used for extracting header information from RPM files. Currently shows package:
- Name
- Version
- Release
- Architecture
- Dependencies
- What gets installed

## Installation 
```
git clone https://github.com/sasutpasut/rpm-header-reader.git
cd rpm-header-reader
make
```

## Usage
```
./rpm-header-reader path/to/package.rpm
```
